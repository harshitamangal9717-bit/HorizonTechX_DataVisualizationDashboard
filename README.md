# Student Performance Data Visualization Dashboard

## Project Overview

This project is completed as part of the Horizon TechX Artificial Intelligence & Data Science Internship Program.

The project is a Data Visualization Dashboard built using Streamlit. It visualizes student academic performance data using interactive charts, filters, KPI cards, and data tables.

## Objective

The objective of this project is to convert raw student performance data into an interactive dashboard for better decision-making and storytelling.

## Tools and Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- VS Code
- GitHub

## Features

- Interactive dashboard interface
- Gender filter
- Result filter
- Grade filter
- Total Students KPI card
- Pass Students KPI card
- Fail Students KPI card
- Average Score KPI card
- Pass vs Fail chart
- Grade Distribution chart
- Average Score Distribution chart
- Test Preparation vs Average Score chart
- Subject-wise Average Score chart
- Math vs Reading Score chart
- Correlation Heatmap
- Top 5 Students table
- Filtered Dataset table

## Dataset

The dataset used in this project is the cleaned student performance dataset generated from the previous EDA task.
