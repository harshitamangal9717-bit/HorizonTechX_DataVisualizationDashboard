# Student Performance Data Visualization Dashboard Report

## 1. Introduction

Data visualization is an important part of Data Science. It helps convert raw data into meaningful charts, dashboards, and insights. In this project, a student performance dashboard was created using Python and Streamlit.

## 2. Objective

The objective of this project is to build an interactive dashboard for analyzing student academic performance.

The dashboard helps users understand performance patterns using filters, KPI cards, charts, and tables.

## 3. Tools and Technologies

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- Visual Studio Code

## 4. Dataset Description

The project uses a cleaned student performance dataset. The dataset includes student details such as gender, parental education level, lunch type, test preparation course, math score, reading score, writing score, average score, result, grade, and performance category.

## 5. Dashboard Features

The dashboard includes:

- Sidebar filters
- KPI cards
- Pass vs Fail visualization
- Grade distribution chart
- Average score distribution
- Test preparation analysis
- Subject-wise average score
- Math vs Reading scatter plot
- Correlation heatmap
- Top 5 students table
- Filtered dataset table

## 6. Key Insights

- The dashboard provides a quick overview of total students, pass students, fail students, and average score.
- Grade distribution helps understand academic performance levels.
- Test preparation analysis shows how preparation can impact student scores.
- Subject-wise visualization compares performance in math, reading, and writing.
- Correlation heatmap helps identify relationships between numerical features.

## 7. Conclusion

This project successfully builds an interactive data visualization dashboard using Streamlit. It helps present student performance data in a simple, visual, and decision-friendly format.
